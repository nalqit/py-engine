# Contributing to PyEngine 2D

Thank you for your interest in contributing to PyEngine 2D!

## How Contributions Work
- Contributors cannot push directly to the main repository.
- All changes must be made via a Fork and Pull Request.
- The project owner reviews all Pull Requests before merging.

## Contribution Steps
1. Fork the repository
2. Clone your fork locally
3. Create a new branch for your change
4. Make your changes with clear, readable code
5. Commit with a descriptive message
6. Open a Pull Request explaining what you changed and why

## Guidelines
- Keep code readable and well-structured
- Follow the existing engine architecture
- Avoid large refactors without prior discussion
- Prefer clarity over premature optimization

## Notes
- This project is experimental and under active development
- APIs may change without notice
- By contributing, you agree that your code will be licensed under the MIT License
